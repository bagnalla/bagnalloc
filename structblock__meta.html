Detailed Description

The structure at the beginning of every block node in the heap (whether free or allocated).

Field Documentation

length

Length of the data portion of the block (doesn't include sizeof this structure).

next

A pointer to the next block. NULL if the current block has been allocated (not free). If the current block is free, next points to either the next free block or end_brk if the current block is the last free block.

prev

A pointer to the previous block. NULL if the current block is the first block.
